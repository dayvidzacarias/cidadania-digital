# Portal de Cidadania Digital e Inteligência Artificial 🌐🤖

Este é um portal de conscientização comunitária desenvolvido para ensinar cidadãos a identificar mídias manipuladas por Inteligência Artificial (deepfakes) e alertar sobre os perigos da desinformação automatizada na sociedade contemporânea.

O projeto foi construído do zero, utilizando boas práticas de desenvolvimento web moderno com foco em acessibilidade e responsividade completa.

## 🚀 Tecnologias Utilizadas
- **HTML5:** Estruturação semântica da plataforma (`main`, `section`, `form`, etc).
- **CSS3:** Layout moderno utilizando **Flexbox**, variáveis de estilização e **Media Queries** adaptáveis para dispositivos móveis.
- **JavaScript (Vanilla):** Manipulação dinâmica do DOM para o validador de quiz e controle do Modo Escuro.

## 🧠 Prompts de Inteligência Artificial Utilizados (Ética Digital)
Para auxiliar no processo criativo de design e arquitetura do código, os seguintes prompts foram aplicados:
1. *"Gere uma estrutura HTML5 semântica para um portal focado em cidadania digital e identificação de deepfakes contendo cabeçalho, conteúdo principal e formulário interativo."*
2. *"Crie regras CSS responsivas usando Flexbox que adaptem um layout de três colunas para uma coluna única em telas de smartphones."*
